## Governing Convention (Current)

1. **`model_id` is the canonical product identity anchor.**
   Deterministic product ↔ image association is model_id-centered, with structured fields as supporting governance context.

2. **Numeric filenames are acceptable only in governed product/colour branches.**
   Files such as `01.png`, `02.png`, `03.png` are valid when the containing folder path already carries deterministic product/variant meaning.

3. **Filesystem hierarchy is semantic and structured.**
   Folder paths must be created from governed structural axes and field boundaries; they must **not** be generated by blindly splitting `model_id` underscores.

4. **Parent folders carry broad taxonomy and governed axes.**
   Parent levels (brand/gender/collection/product type/sub-type/axis families) define context and constraints for lower levels.

5. **Final product/variant branches should avoid redundant token repetition.**
   Branches near the leaf should not restate brand, gender, collection, product type, or sub-type tokens already encoded in parent folders.

6. **Colour/variant folders remain terminal or near-terminal before media files.**
   Colour/variant should be the final (or near-final for controlled media subfolders) branch before image files.

7. **Runtime/database image path strategy can vary, but must be deterministic and documented.**
   Acceptable strategies include:
   - semantic-path-based
   - model_id-based
   - mapping-based between source/archive semantics and runtime delivery paths

8. **Token-overlap matching is non-authoritative when conflicting with governed identity.**
   If token overlap conflicts with `model_id`, `collection`, `subCategory`, or product type constraints, the governed structured identity wins.

9. **Batch 1 remains operationally valid.**
   Existing Batch 1 operations remain valid unless a later reviewed migration explicitly normalises them.

10. **Batch 2 requires pre-migration convention planning.**
    Before any folder copying or ProductDB/MySQL reconciliation for Batch 2, execution must be planned against this convention and reviewed.
